请输入替换字段的列表，每行为一条记录，逗号左侧为被替换的字符串，逗号右侧为要替换成的字符串






使用说明:
一键分身功能为神农系统的招牌功能，通过设置替换列表，则源文件夹中的所有的文件中“类似项目标识”的字符串均会替换掉
例如:
SN,FGATE这个配置，将会把SN_parent中的SN全部替换掉，而SNAPSHOT中的SN不会替换掉
技术大致原理是：关键词左侧为非字符(大于小于回车换行等)且右侧为非字符(大于小于回车换行等)，则需要替换掉

通过此功能，我们可以做到分身：
1、迅速新建一个新项目并立即开始
2、将已经分身出去的项目通过反向操作合并回来
就可以实现通过一个系统，幻化出N多完全系统，如果新系统有好的修改，则通过分身工作可以把分身收回来，保持神农系统的继续进化。
